# Pantry

냉장고 재고 관리 및 재해 대비 재고 관리를 위한 Flutter 앱입니다.

## 주요 기능

- 🥶 냉장고 재고 관리 및 유통기한 추적
- 📦 재해 대비 재고 관리
- 📸 OCR을 통한 레시피 스캔
- 🗺️ 주변 피난소 위치 확인
- 🔔 유통기한 및 재고 알림

## 기술 스택

- Flutter 3.10.3+
- Riverpod (상태 관리)
- go_router (라우팅)
- Firebase (Firestore, Firebase AI)
- Google ML Kit (텍스트 인식)
- Google Maps Flutter

## 빠른 시작

```bash
# 의존성 설치
flutter pub get

# 앱 실행
flutter run
```
